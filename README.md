# react-native-call-detector

This library is based off of @huddle01@react-native-call-detection for detecting call states

## Installation


```sh
npm install react-native-call-detector
```


## Usage


```js
import { multiply } from 'react-native-call-detector';

// ...

const result = multiply(3, 7);
```
